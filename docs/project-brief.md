# Project Brief: Dan Sucks Fart Button

## Goal
Build a web page that says "Dan sucks!" and has a button that plays a fart noise.

## Requirements
1. Create a browser-viewable static page.
2. The page must visibly include the exact phrase: `Dan sucks!`.
3. The page must include a clickable button.
4. Clicking the button must play a fart-style sound effect.
5. Prefer Web Audio synthesis over committed binary/audio assets unless there is a strong reason otherwise.
6. Keep it simple, local-first, and safe for a throwaway Claw Labs gag prototype.

## Success Criteria
- A local browser can open the page.
- The headline is visible.
- The button is visible and clickable.
- Clicking the button produces the sound effect.
- Browser console has no JavaScript errors after loading and clicking.

## Implementation Notes
- Plain HTML/CSS/JavaScript is enough.
- No package install should be necessary.
- Use `index.html` at the repository root unless Paperclip has a better minimal structure.

## Boundaries
- Do not deploy or publish publicly.
- Do not use paid/external services.
- Do not request secrets.
